SUONI PER L'APP PICTURE CHECK

In questa directory devono essere presenti i seguenti file audio:
1. affermazione.mp3 - Suono da riprodurre quando le foto devono essere fatte
2. negazione.mp3 - Suono da riprodurre quando le foto sono già state fatte

Nota: I file audio devono essere in formato MP3 per garantire la compatibilità con tutti i browser.
